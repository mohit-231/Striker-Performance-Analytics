# Striker Performance Analytics and Classification Framework

## Project Overview
This project focuses on analyzing football striker performance data using data analytics and machine learning techniques. The objective is to evaluate player performance, identify key performance indicators (KPIs), and classify player performance based on statistical metrics.
The project demonstrates practical applications of data cleaning, exploratory data analysis (EDA), visualization, and predictive analytics.

---

## Key Features
- Player performance analysis
- Data cleaning and preprocessing
- Statistical analysis and visualization
- Performance classification using machine learning
- KPI-based analytical insights
- Data-driven player evaluation

## Dataset Information
The dataset contains football striker performance metrics such as:
- Goals scored
- Assists
- Matches played
- Shots on target
- Performance statistics

Dataset used:
- `Strikers_performance.xlsx`
